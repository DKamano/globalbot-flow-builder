# Globalbot Flow Builder

Ferramenta CLI desenvolvida em Node.js para padronização da criação de fluxos de chatbot.

## 📌 Projeto Integrador – UNISA

Este projeto tem como objetivo apoiar a equipe de Customer Success na criação estruturada de fluxos de chatbot, reduzindo inconsistências, retrabalho e tempo de onboarding.

## 🚀 Funcionalidades

- Coleta guiada de informações
- Validação automática do fluxo
- Geração de estrutura em JSON
- Geração de relatório em TXT

## 🛠️ Tecnologias

- Node.js
- JavaScript

## ▶️ Como executar

```bash
node index.js
